削增益魂武

魂武

系统设置

  1. 建议使用动作模式，或者普通模式在设置中打开技能自动选择目标释放。

技能介绍

姿态

  1. 每次切换姿态给予相应的增益，战斗中切到矮人给队友5目标1层稳固；切到恶魔给对队友5目标3秒抗性。
  2. 每次切换姿态会触发超级能量法印``超级力量法印等效果，等同于战斗中切换武器。
  3. 矮人姿态下使用F2，恢复25点能量，且给予自身50%硬减伤。
  4. 恶魔姿态下使用F2，恢复25点能量，且给予自身3秒抗性。
矮人
  1. 治疗技能，清除症状，并治疗。
  2. 稳固激发，核心输出技能，一条直线上按段造成伤害并提供脉冲式的稳固。
    • 目标数几乎可以打满全团，只要对面前后站成一条直线。
    • 经常在打进攻节奏时使用，造成伤害的同时为身后的队友提供持续的稳固。
    • 配合大锤2使用，形成一波直线爆发。
  3. 复仇之锤，近身肉搏神技能，为自己提供减伤，伤害敌人并且治疗自己
    • 持续对周身造成伤害，几乎没有目标上限，也就是说在人堆里，可以造成多目标多次数大量伤害。
    • 持续提供13%的无法驱散的硬减伤。
    • 持续治疗自身，恢复量不多但是在最终肉搏阶段依然非常可观。
    • 配合 剑2使用，对周身造成一波脚下爆发。
  4. 嘲讽， 远距离多目标嘲讽，多配合进攻节奏使用来配合控制。
  5. 矮人精英，提供50%硬减伤并解控。
    • 开场被控，请毫不犹豫的使用它来解控。（开场大多数情况下能量满的）
    • 优先级低于石板路。
恶魔
  1. 治疗技能，吸收症状，并治疗。症状越多回血越多
  2. 吸收痛苦，几乎不用
  3. 恶魔跳， 位移+群体拉拽，多在脱团时调整位置使用，它会消耗你很多的能量。需要大量控制时也可以使用。
  4. 驱除附魔，BD核心技能 消除敌方增益。
    • 有恶魔能量优先供应它，即F2只在恶魔姿态下使用，恶魔姿态下尽量只使用驱除附魔，其他技能不用。
    • 使用一次最多可以消除目标及到目标连线上，5目标，每个目标3个增益。（5x3=15)

武器技能

大锤
  1. 大锤2，1200码直线三段式5目标，大量伤害。
    • 伤害有延时，需要一定的预判。
    • 无法打穿建筑物拐角，或者有一定坡度的上下坡楼梯等。
  2. 大锤3，1200码区域内5目标，大量伤害。
    • 常配合大锤2使用，可以让大锤2第3段伤害和大锤3伤害几乎同时击中目标，造成秒杀效果。
    • 指挥不叫起跳不要用，在移动中使用会掉队，在后撤中使用会被卖。
    • 大锤3 飞行过程中全闪避，可以用它来远距离触发敌人的陷阱埋伏。
  3. 大锤4，几乎不用
  4. 大锤5，1200码区域击倒，控制技能
剑剑
  1. 剑1，近身肉搏时伤害可观
  2. 剑2，近身爆发
  3. 剑3，闪避+位移+多段伤害，谨慎使用，它可能会使你脱团。
  4. 剑4，伤害+定身，配合打近身爆发
  5. 剑5，位移+伤害，谨慎使用，它可能会使你脱团。可以用它来压在非集火区域的尸体，二段造成巨量伤害。

进阶操作

  1. 熟练使用解控+翻滚来摆脱困境
    • 矮人精英 + 翻滚
    • 吸收痛苦 + 翻滚
  2. 锤2+石板路直线爆发，以嘲讽衔接控制

    大锤2->石板路->嘲讽

  3. 剑2+ 复仇之锤近身爆发，以剑4衔接控制

    剑4->剑2->复仇之锤

  4. 治疗技能，不要省cd好了就交，甚至可以满血提前交，触发吸收投掷物的护盾效果
  5. 切武器，切姿态的时机
    • 肉搏切双剑，非肉搏尽量大锤
    • 理论上切姿态，CD好了就切，不用顾及剩余能量
    • 切到矮人状态优先使用石板路，等下一次切到恶魔之前，再使用一次石板路。
    • 切到恶魔姿态尽量只使用驱除附魔。
    • F2回复能量也只在恶魔姿态使用增加削增益量
