辅助燃火

燃火

系统设置

  1. 在“常规选项/用户界面”下，选中始终显示团队血条和始终显示队伍血条。更宽的队伍血条和更宽的团队血条取决于个人喜好设置，一般而言小队建议设置更粗的血条。
  2. 进团之后，在“队伍选项”中，取消选中“以网格模式浏览”。能够更加方便的跟踪盟友的增益，症状和血量，从而实现更好的决策和技能管理。

技能介绍

通用技能

坚守阵地
  1. 坚守阵地瞬发，能够使小队获得5层10s+的稳固，使其成为燃火核心辅助技能。
  2. 坚守阵地的稳固持续时间很长，这是小队稳固的主要也是最重要的来源。为了队友不会被控制，需要预判在即将接战之前2-3s使用。
  3. 如果已经被控制技能控制了，除非绝对必要，否则请勿使用坚守阵地解除昏迷,而使用解放咒语。
神圣之地
  1. 大范围放置类稳固区域，持续效果最多可叠加到3层稳固。
  2. 技能需要读条且放置类技能，需要提前预判团队走向向车头前方释放，或者在团队抱团站定时使用。
庇护之地
  1. 放置类偏防御控制技能，能够把没有稳固的想穿过他的敌人弹飞，敌人穿过他至少需要2层稳固。
  2. 技能需要读条且放置类技能，需要提前预判地方团队走向，且敌方要突脸或者对穿时放置在他们进场的位置上。
  3. 该技能拥有吸收投掷物和脉冲治疗效果，也可在对峙时释放在己方大团脚下。
反射之墙
  1. 超宽范围180度面向反射投掷物，一般放置在己方大团跟前。
  2. 仅建议在攻城战时，或者敌方团队为狼群打法时使用。

圣书

  1. 激活后，所有圣书都会保留被动，因此如果团队压力很大，请不要犹豫使用它们，可以直接从圣书切换到另一个圣书。
  2. 进入圣书或者退出圣书时会提供短时间的急速，合理利用它们可使一些读条技能快速释放，例如：
    • 可以更快地在进入圣书之后第一时间使用出圣书4和圣书5
    • 或者在推出圣书时读条法杖4。
  3. 如果无法使用掉所有书页，那么尽快的切出圣书使其进入冷却时间。
  4. 圣书4和圣书5通常是最强的，但cd时间最长。在可能的情况下，仅在一次进入圣书时，请尝试至少使用两次圣书4和圣书5，然后再用尽书页。
  5. 战斗中无论使切入还是切出都可触发超级能量法印效果
正义之书
  1. 优先级较低的填充技能，除非没有其他选择，否则不会被大量使用。使用场景：
    • 需要快速控制圣书3
    • 需要触发超级能量法印
    • 团战结束收包阶段使用
  2. 圣书5 将用于增加群体伤害
  3. 圣书4 对穿的时候用
  4. 圣书3 一个很好的控制技能。
  5. 圣书1 填充技能600范围的扇形输出。
决心圣书
  1. 决心圣书带来了高水平的治疗效果，症状净化，症状转增益以及为您的小队提供独特的增益(结语：永恒的绿洲)。
  2. 圣书5 尽量用此技能作为第一个和最后一个圣书技能，以提高队友的治疗效果（尤其有用，您的队伍有另一个奶的时候）。当经过集火区域或被敌人贴脸战斗时，它也可以用于清症。
  3. 圣书4 是优先技能，通常是进入圣书后第二个技能。由于它是静态治疗水圈，请尝试将其施放在车头位置，或者如果队伍正在移动，则将其放到指挥官的前方。
  4. 圣书3 一般不使用。
  5. 圣书2需要进行适度清症时使用。尽量使用在开放通道不可用时候。由于半径很小240，尽量抱团时使用。
  6. 圣书1 如果判断团队承受大量的伤害，则可以使用它作为填充治疗技能。它的作用范围600码扇形区域，所以治疗在安全距离之外对掉队的队友也是可以使用的。

最优使用顺序: 决心圣书=>圣书5=>圣书4=>圣书1x4=>圣书4=>圣书5=>法杖4/盾牌4 最快使用顺序: 决心圣书=>圣书5=>圣书4=>圣书1x6=>法杖4/盾牌4

勇气之书
  1. 勇气之书为您的团队提供了非常强稳固增益支持，队友提供了独特的被动奖励。此圣书的优先级高于决心圣书。
  2. 圣书5通过韧性赋予队友一些防御护甲，使DPS生存能力更好。尝试将其用作第二个和最后一个圣书技能，或者在进行push或接团时使用。
  3. 圣书4 是对抗症状压力的主要技能，也是BD中核心技能之一。
    • 该技能可以为大面积半径内的队友提供抗性，消除症状的影响，特别是对残废，定身和冰冻这类软控作用巨大，避免队友脱离团队。
    • 在进攻或撤退时使用(假设抱团不动作用不大，简单来讲关键的团队转移时机就是使用它的最佳时机)，以确保被困在后面的盟友接收到抗性脉冲。
    • 由于它是静态+脉冲的，请尝试将其释放在车头位置，或者，如果指挥官正在移动，则根据他的移动将其稍微向前移。
  4. 圣书3 仅用于反射投掷物，看情况释放。它可以放置在敌我双方团队之间，以确保反射出所有射出的投掷物。
  5. 圣书2 是低优先级的圣书技能，嘲讽周围目标。仅应在近战范围内，与对方进行对穿时使用。
  6. 圣书1 不要盲目使用1技能，请使用它来补稳。由于它具有600射程，因此可以用来在安全距离外向后方支援盟友。
  7. 始终检查您的小队的增益，以确保没有浪费圣书页，没有被浪费的增益。例如施放圣书4，当团队脚下没有几个红圈，或者你的小队没有任何症状时。

最优使用顺序 勇气圣书=>圣书4=>圣书5=>圣书1xN=>圣书4=>圣书5=>法杖4/盾牌4

咒语

  1. 燃火站位偏团队后方后面，以确保您的盟友受到咒语技能的影响。
  2. 咒语是瞬发三层充能，确保咒语一直进入充能计时，以确保不会浪费充能时间。被控制后仍然可以使用咒语，所以请选择好咒语的释放时机。
  3. 治疗咒语是你的治疗技能。它可以提供大量的圣盾。 圣盾对于消除直接的爆发伤害至关重要，当你想要摸起被击倒的队友时，可以使用治疗咒语为自己和他们提供圣盾。
  4. 清症咒语 AoE清症技能。当团队需要清除轻微/中度的症状，就可以使用它。时刻检查自己的盟友是否有症状压力，这样就不会浪费咒语次数。
  5. 稳固咒语 是随需应变的AoE解控和稳固技能。是游戏中少有的为队友解除控制的技能。 通常在需要较少层数的稳固或突发情况的情况下使用。如果您不确定敌人的大团是否会推进，或者如果他们只是假性地推进骗技能，建议对队友使用此技能（以及治疗咒语）。

法杖

  1. 法杖拥有扎实而灵活的1200射程技能，可以在进攻方面提供威能和地面控制，也可以在防御方面提供治疗和清症。
  2. 神圣打击 提供爆发AoE恢复。由于它需要读条，请预判位置释放。神圣打击适用于轻微症状清除或少量的治疗压力，优先级比较高的技能。建议绑定切换武器以正确使用此技能(即每次切换到法杖武器组，第一时间使用神圣打击，或者从法杖武器组切到其他武器组或者圣书时，优先使用神圣打击之后再切换技能)。另外它是爆炸技能，如果可能的话，请尝试将其释放在组合区域之上，尤其是在流水区域。
  3. 迅捷符印在团队开局抱团刷增益时使用它，以堆叠友军迅捷。当指挥官要求炸烟、炸火时，请不要用它，除非你想指挥喷你
  4. Empower 确保队友在战斗前获得12层威能，这对于团队的输出至关重要。Empower读条结束时会提供强大的治疗效果，而不仅应被用作战斗前增强技能。在战斗中，每当切换到法杖武器组或退出圣书到法杖武器时都要使用它。请记住，此技能需要很长的读条时间，这意味着您将停留相当长的时间。请选择好释放的时机，如有必要，请随时准备切换武器以将其取消读条。
  5. 法杖5 无法躲闪的界限类技能，这意味着没有稳固的敌人即使躲闪也会被击倒。使用小技巧，如果您发现这条线释放位置有误，敌团不会经过预定释放的位置，那么你可以切换武器取消释放。

斧/盾

  1. 斧/盾将盾牌的防守支持和斧头AoE拉动，与“近战球”的游戏风格完美融合。
  2. 斧2 残废近距离的敌人，一般在炽烈之刃拉到敌人之后使用。也可以用给队友提供激怒增益。
  3. 炽热之刃 有效地削增益和罩子配合把没有稳固的敌人拉到身前。
  4. 盾4 主要提供圣盾和保护，尽量cd好了就用。
  5. 盾5 可以使用它击退敌人。此外，通过双击使用以进行大范围的突发AoE治愈或引导护盾防御投掷物。

进阶操作

  1. 决心圣书和勇气圣书两者都带来了新的技能组合，可为团队提供极为强大的增益支持，治疗和清症以及独特的增益。
  2. 知道何时进入或退出一本圣书，必须根据团队情况以及在圣书处于冷却状态具体处理。
  3. 在圣书使用之外以及在圣书使用之间，合理穿插使用武器技能以及咒语和通用技能，将为团队提供更加强大的支持。
  4. 翻滚治疗和纯洁的心占整个团队恢复的重要部分。不仅在个人防御上，而且在小队成员需要恢复时，也要使用翻滚治疗和圣盾技能。也就是说翻滚即可以给队友治疗，超级能量法印对于燃火来说是必须带的。
  5. 记住，进/出圣书和交换武器都会触发超级能量法印效果。翻滚就可以治疗队友，那么能量条=治疗。关键时候可以原地来回翻滚来规避伤害+恢复。
  6. 读条法杖4的同时，给自己一个治疗咒语，尝试触发急速来加快法杖4的读条。
