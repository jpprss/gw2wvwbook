系统设置

技能介绍

通用技能

净化机械
  1. 移动清症，瞬发，光圈。
  2. 此技能使用5秒后会给予包括自身5目标的奔驰疾速BUFF，维持3秒，小奔驰指的就是此类技能，同样效果的还有壁垒机械
  3. 净化机械腰带技能→化学力场：瞬发，原地释放一个毒圈，释放优先级不高，可以考虑不释放本技能。
壁垒机械
  1. 小奔驰第一优先级，提供的奔驰疾速BUFF原理同净化机械，建议新手只拿此技能当做奔驰技能来用。
  2. 壁垒机械可以提供护盾，并且看起来护盾量还不少，但得到护盾的代价是分摊队友伤害。遇到人数多、强度高的团，尤其是在敌方集火时开启此技能，血线会极不稳定，如敌方集火时不小心开启此技能，建议马上翻滚规避伤害。
  3. 壁垒机械腰带技能→防御立场：提供3层稳固持续9.5秒，格挡飞弹。把此技能当成一个补稳固技能而不是格挡技能，第一次释放时机在指挥喊过“一稳”后3-5秒左右，后面建议好了就放。
R型试剂
  1. 解除自己定身，获取100耐力，一般两连翻滚后使用
  2. R型试剂腰带技能→投掷R型试剂：投掷一个光圈，可以给倒地状态的队友抬血，配合指挥指令使用，指挥口中的R药就是它。
    • 持续时间10秒，在持续时间内同一人反复倒地可一直享受R药抬血效果。
    • 本技能CD长达90秒，投掷距离远可达1200码，但光圈极小半径只有180 码，几乎每一波团战只有一次使用机会，看清尸体位置，注意不要放偏。
空投补给
  1. 给敌人造成昏迷效果，释放时机是在双法池时配合控制效果。
  2. 三炮台落地时产生一次组合技“爆炸”效果，手动引爆时再次产生“爆炸”效果，可以配合炸圈。
功能机型
  1. 可以抬尸，压尸，提供奔驰疾速BUFF。
  2. 释放后立刻给与3目标队友持续3秒的奔驰疾速效果。
  3. 向倒地队友释放此技能可以让器械摸队友，拉人。
  4. 向倒地敌人释放可以快速压尸，3秒插旗。
  5. 压尸抬尸时小机器人会被打爆，如尸体位置被敌方火力覆盖，建议放弃使用，等待下一次机会。

武器技能组

大锤

  1. 1技能：基本不用

  2. 2技能：读条期间反弹飞弹，造成组合技“旋风”，可以打出两发组合技之矢。本技能只反弹飞弹不等于反弹其他伤害，读这个技能≠读无敌技能。

  3. 3技能：扣掉，避免使用。如不小心误按，马上按医疗包（或试剂枪）键位+切武器键位，可以停下锤3位移。

  4. 4技能：读条时格挡，击中敌人给予自身屏障，自保技能。

  1. 5技能：释放距离1200码，目标数量？只在第一次脉冲产生昏迷效果，持续1秒，每次集火时尽量使用。本技能只有第一波脉时产生昏迷效果，也就是说只有直接击中敌人，才会产生昏迷，敌人第一次踩到本技能≠敌人吃到第一波脉冲。
枪盾
  1. 1技能,2技能：基本不用
  2. 3技能：瞬发，有目盲和困惑效果, 一般不用。

  3. 4技能：提供保护BUFF，读条可以反射敌方投掷物，再按一次结束读条，产生组合技“爆炸”效果并击退敌人。

    • 强烈建议不读条直接引爆, 即双击来击退敌人。

  4. 5技能：提供保护BUFF，读条时遭受敌人攻击会格挡并回馈攻击者昏迷BUFF。再 按一次会扔出盾牌对途经敌人造成晕眩，扔出的盾牌属于投射物。

    • 强烈建议只读条不扔盾牌，即单击来使攻击者昏迷。

医疗包技能组

  1. 1技能：对着队友一直按不要停，1技能提供的治疗量并不少。此技能范围600码，有效距离远，从车尾喷到车头毫无问题。
  2. 2技能：面向队友，好了就放。每个绷带都单独有治疗量，比如一个残血的输出队友和你站的足够近，吃到2技能全部绷带是可以直接奶满的，有爆发奶人效果，但条件苛刻，需要距 离很近，知道这个原理就行，打团时面向队友放，不用考虑太多。
  3. 3技能：原地放一个水圈，有清症效果，好了就放。当指挥喊"大水/炸水"时需使用，不用刻意留此技能。当前BD下只有这一个水圈技能。
  4. 4技能：群体治疗，有效范围300码，抱团时使用最佳。
  5. 5技能：爆炸技，有组合技爆炸效果，指挥喊“炸水”时使用。
  6. 自我包扎： 医疗包腰带技能，默认F1按键，自身大量回血，给与自身和队友奔驰疾速。指挥口中的"大奔驰"就是它，大团入场前释放，正常指挥会下指令，切记第一波配合指令，不要随意乱放。BD中点出“分散治疗”后，此技能由自身回血变为群体回血。

试剂枪技能组

  1. 1技能，2技能：投掷物，基本不用。
  2. 3技能：有强力且爆发性的清症效果，奶工清症灵魂技能，好了就放，释放时面向自家队友。释放此技能前一定要注意，不要选中敌人作为目标，不然此技能会跟着敌人位置移动而改变面向。
  3. 4技能：新手请扣掉这个技能。
  4. 5技能：加血，清症，光圈，对着人群，好了就放。
    • 此技能只在落地时产生一次清症效果，剩下的脉冲全部为加血脉冲，无清症效果，当成加血技能来用就好。
    • 此技能飞行速度慢，同样是落地才产生效果，记得预判。
  5. 治疗喷雾：试剂枪腰带技能，瞬发，解除昏迷，提供再生BUFF，唯一一个解控技能，不被控制永远不要按。

进阶操作

  1. 医疗包爆发治疗: 医疗包5 -> 医疗包3 -> 医疗包4
  2. 试剂枪4炸圈：原地产生组合技“爆炸”效果，并向人物面向的相反方向跳跃位移550码，跳跃过程中僵直，僵直时无法释放技能及翻滚，跳跃落地前全程僵直。技能读条结束人物开始跳跃位移期间，按下“切换武器”键可以中断位移，但无法中断跳跃动作以及跳跃产生的僵直，需要大量练习去熟悉此技能手感。
  3. 指挥下“对爆”“魂武起跳”等远程消耗指令时，马上向敌方大团交出大锤5技能进行破格挡，注意一定要把技能交到最远距离。交完锤5后切医疗包放3技能原地清症，读完3技能持续读1技能。
  4. 指挥下“小奔驰”指令时，马上交出腰带5技能功能机型给到自己脚下，提供5目标3秒钟的疾速奔驰。
  5. 指挥下“大奔驰”指令时，马上交出腰带1技能自我包扎，切进医疗包技能准备奶血。
  6. 提前开净化机械技能，随后医疗包2、3、4、5技能啥亮按啥，全按进CD切试剂枪
  7. 切入试剂枪优先开启试剂枪3技能，读完条接试剂枪5技能，放完5技能切进武器技能组。
  8. 切入武器技能组，优先看锤5技能冷却时间，如果CD好了立刻砸向敌方大团后切医疗包读冷却好的技能；如果CD没好则读锤2并切入医疗包读冷却好的技能。
  9. 解控技能第一优先交治疗喷雾，解控后马上接翻滚防止二次被控。
  10. 医疗包主要负责治疗，在医疗包技能组理应停留时间最多，切其他技能组时要明确自己想干什么，在其他技能组停留的多余时间越短，治疗收益越大。在医疗包技能组时，1技能需要一直按，即使是读其他技能，1技能也要不停的按。
